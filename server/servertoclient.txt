1. 로그인에 대한 응답
    1) 로그인 성공 : @login 0 		//로그인이 성공하였습니다.
    2) 로그인 실패 : @login 1		//로그인이 실패하였습니다.
    3) 중복 로그인 : @login 2		//다른 곳에서 이미 로그인 하였습니다.
    4) 다시 로그인 : @login 3		//현재 로그인 한 클라이언트에서 다시 로그인 시도

2. 로그아웃에 대한 응답
    1) 로그아웃 완료 : @logout 0	//로그아웃 하였습니다.
    2) 로그인 상태 아님 : @logout 1	//로그아웃 상태입니다.

3. 아이디 찾기에 대한 응답
    1) 입력한 정보 일치 : @findid 0,아이디
    2) 입력한 정보를 찾지 못함 : @findid 1	//해당 아이디를 찾을 수 없습니다.

4. 비밀번호 찾기에 대한 응답
    1) 입력한 정보 일치 : @findpass 0		//해당 메일로 비밀번호를 보내드렸습니다.
    2) 입력한 정보를 찾지 못함 : @findpass 1	//잘못된 정보를 입력하셨습니다.

5. 사용자 등록에 대한 응답
    1) 등록 성공 : @join 0		//정상적으로 등록이 완료되었습니다.
    2) 계정 중복 : @join 1		//같은 아이디의 계정이 존재합니다.
    3) 등록 불가 : @join 2		//로그인 한 사용자가 계정등록을 시도하는 경우

6. 중복 아이디 확인에 대한 응답
    1) 입력 아이디 찾지 못함 : @dupid 0	//사용가능한 아이디입니다.
    2) 입력 아이디 존재 : @dupid 1	//사용할 수 없는 아이디입니다.

7. 점수 확인에 대한 응답
    1) 기록이 존재 : @record 0,YY-MM-DD,점수/YY-MM-DD,점수/YY-MM-DD,점수/.....
    2) 기록이 없음 : @record 1		//기록이 없습니다.
    3) 로그인 하지 않음 : @record 2	//로그인하지 않으면 기록을 볼 수 없습니다.

8. 오늘 기록 여부에 대한 응답
    1) 오늘 기록 존재 : @today 0	//오늘 점수가 이미 기록되어있습니다.
    2) 오늘 기록 없음 : @today 1	//오늘 새 기록에 도전하세요.
    3) 로그인 하지 않음 : @today 2	//로그인을 하지 않은 상태이므로 확인할 수 없습니다.

9. 점수 저장에 대한 응답
    1) 점수 저장 성공 : @save 0		//점수가 정상적으로 저장되었습니다.
    2) 로그인 하지 않음 : @save 1	//로그인을 하지 않은 상태에서는 점수를 저장할 수 없습니다.

10. 게임별 수준(나이) 반환에 대한 응답
    1) 반환 성공 : @average 0,점수0,점수1,점수2
    2) 로그인 하지 않음: @average 1

11. 사용자 정보 반환에 대한 응답
    1) 정보 반환 성공 : @getinfo 0,이름,메일주소,나이
    2) 로그인 하지 않음: @getinfo 1

12. 사용자 정보 수정에 대한 응답
    1) 정보 저장 성공 : @setinfo 0
    2) 비밀 번호 틀림 : @setinfo 1
    3) 로그인 하지 않음: @setinfo 2

13. 탈퇴에 대한 응답
    1) 탈퇴 성공 : @leave 0
    2) 로그인 하지 않음: @leave 1
    3) 비밀번호 틀림 : @leave 2
