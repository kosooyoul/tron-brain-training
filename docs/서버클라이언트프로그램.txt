<서버프로그램>

1. 서버프로그램 시작
2. 소켓통신을 하기위한 포트번호를 설정, 윈소켓을 활성화
3. 클라이언트로부터의 요청메시지와 접속요청을 기다림
4. 클라이언트가 서버에 접속, 다수의 클라이언트가 접속시 클라이언트 수 만큼 윈소켓을 복사하여 TCP/IP로 통신
   (클라이언트가 접속을 종료할 경우 해당 클라이언트와 통신한 윈소켓은 언로드)
5. 클라이언트로 부터 수신된 요청메시지(#로 시작)는 큐에 공간이 없을 때까지 큐에 저장
6. 큐에 저장된 메시지는 메시지처리 프로시져를 통하여 읽혀지게됨
7. 메시지처리 프로시져는 메시지의 형태를 분류하여 해당되는 루틴 또는 함수를 수행
   (각 함수는 메시지 형태에 따라 데이터베이스에 접근하여 레코드를 읽고, 삽입, 수정, 삭제를 수행)
8. 분류된 메시지를 처리한 함수는 처리결과를 지정된 응답 메시지형태(@로 시작)로 만들어 반환
   수신된 메시지의 형식이 잘못되었을 경우 오류를 알리는 응답 메시지형태로 반환
   해당 메시지를 보낸 클라이언트가 메시지 처리도중 접속을 종료했을 경우 무시
9. 반환된 응답메시지는 윈도우소켓을 통하여 클라이언트에 전송
10.3번과 같이 클라이언트로부터의 요청메시지와 접속요청을 기다림


<클라이언트>

1. 클라이언트 시작 및 서버에 접속(서버IP와 포트번호는 컴파일 전에 설정)
2. 클라이언트는 로그인이나 사용자등록, 아이디 및 비밀번호 검색 기능을 사용
3. 서버에 사용자 등록을 하지 않은 클라이언트는 아이디와 비밀번호, 이름, 나이 그리고
   비밀번호 분실시 서버로부터 비밀번호를 수신받을 이메일 주소를 입력하게 사용자 등록을 하게됨
   아이디 입력시 아이디 중복여부확인이 가능하며, 비밀번호는 두번 입력받아 실수를 방지
   (각 텍스트필드에 입력한 정보는 요청메시지형태로 조합하여 소켓으로 서버에 송신)
   (입력받은 정보가 잘못되었거나 중복아이디인 경우 사용자 등록이 실패되었음을 알림)
4. 아이디 혹은 비밀번호를 분실한 클라이언트는 필요한 정보를 입력
   아이디는 이름과 이메일 주소로 검색하여 화면에 출력
   비밀번호는 아이디와 이메일 주소로 검색하여 해당 사용자의 이메일 주소로 비밀번호를 발송
   (아이디와 비밀번호를 찾지 못했을 경우 검색이 실패되었음을 알림)
   (비밀번호 검색후 비밀번호는, 이메일 발송함수를 작성한 웹상의 PHP프로그램에 접속, 매개변수로 전달되어 해당 사용자의 이메일 주소로 발송)
5. 아이디와 비밀번호를 입력하여 로그인
6. 계산능력(테스트번호 = 0), 암기능력(1), 공간지각능력(2)의 3가지 테스트와 점수확인 선택이 가능
7. 점수확인은 해당 사용자의 계산능력, 암기능력, 공간지각능력 점수 및 기록날짜를 확인가능
   (선택한 테스트번호로 요청메시지를 조합하여 소켓으로 서버에 송신)
   (서버로부터 수신받은 응답메시지를 읽어 각각의 날짜와 점수로 분리하여 화면에 출력)
8. 계산능력 테스트는 임의로 발생된 두자리의 자연수 2개를 더하는 형식의 10문제를 출제하며 제한시간은 10.0초로 설정
   입력된 답(y)는 초기 0으로 표시, 숫자버튼(해당 버튼의 숫자:x) 누름과 동시에 y = y * 10 + x 로 입력값을 갱신
   제한시간이 카운트 다운되어 0.0초가 되었을 경우 입력값이 답인지 확인하여 정답여부를 표시
   정답인 경우 정답 횟수를 증가하여 10번째 문제까지 완료한 후 정답 횟수로 점수를 계산
   계산된 점수와 해당 테스트 번호(0)로 요청메시지를 조합하여 소켓으로 서버에 송신하여 서버의 데이터베이스에 점수를 저장
9. 암기능력 테스트는 0~9의 숫자 10개를 임의로 섞어 화면에 표시하면 사용자가 10.0초간 암기, 20.0초간 답 입력하도록 하였으며 5문제 출제
   10.0초간 암기하는 동안 숫자버튼 사용이 불가능, 답 입력시 숫자버튼 사용이 가능
   출제된 문제와 입력한 답은 int형 배열로 저장
   암기 10.0초, 답 입력 20.0초, 제한시간 총 30.0초가 카운트 다운되어 0.0초가 되었을 경우 출제된 문제와 입력한 답에서 일치된 숫자의 갯수를 확인
   5번째 문제가 종료되어 총 점수는 총 일치된 갯수로 계산
   계산된 점수와 해당 테스트 번호(1)로 요청메시지를 조합하여 소켓으로 서버에 송신하여 서버의 데이터베이스에 점수를 저장
10.공간지각능력 테스트는 블럭의 수를 세는 테스트로 총 10문제를 출제하며 제한시간은 10.0초로 설정
   입력된 답은 초기 0으로 표시되며 입력방식은 계산능력 테스트와 동일
   블럭은 미리 만들어 놓은 블럭이미지를 임의로 표시
   제한시간이 카운트 다운되어 0.0초가 되었을 경우 입력값과 해당 블럭의 갯수를 저장해둔 값과 일치하는지 확인
   정답인 경우, 계산능력 테스트와 동일하게 10번째 문제까지 완료한 후 정답 횟수로 점수를 계산
   계산된 점수와 해당 테스트 번호(2)로 요청메시지를 조합하여 소켓으로 서버에 송신하여 서버의 데이터베이스에 점수를 저장

*  랜덤값의 발생은 타이머로 발생
